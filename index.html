HTML/GitHub Project

By means of this HTML and GitHub project, I acquired useful knowledge in contemporary web development techniques and version control systems. Building a responsive website from scratch improved my knowledge of semantic HTML5 elements and CSS Grid layouts, thereby matching with Ryou and Ryu (2018) ideas of usability. Working with GitHub helped me to better understand collaborative development techniques, especially the relevance of commit messages and branching strategies as stressed by Constantino et al. (2020).

Key learnings:

  • Implemented responsive design principles using CSS Grid and Flexbox
  • Mastered basic Git workflows including commits, branches, and merges
  • Applied semantic HTML5 elements for better accessibility and SEO
  • Learned the importance of mobile-first design approaches

What I Learned

CARP Project

The CARP project helped me to grasp basic design ideas more deeply. Inspired by Williams' (2015) ideas from "The Non-Designer's Design Book," I investigated how these four components interact to produce cohesive layouts and successful visual hierarchies.

Key insights:

  • Utilized contrast to establish clear visual hierarchies
  • Applied alignment principles for professional presentation
  • Implemented repetition to create visual consistency
  • Leveraged proximity to organize related elements effectively

What I Learned

Typography Project

This project increased my respect of typography as a fundamental component of web design. Inspired by Bringhurst's (2004) ideas of typographic brilliance, I tried several typefaces, spacing, and hierarchies to produce readable and visually appealing layouts. The project clarified for me how typeface shapes brand impression and user experience.

Key discoveries:

  • Explored the relationship between typeface choices and content tone
  • Learned proper spacing techniques for optimal readability
  • Practiced creating effective typographic hierarchies
  • Understanding the importance of responsive typography

What I Learned

Typography Systems Project

Typographic organization has always been a complex system integrating multiple elements such as hierarchy, texture, order of reading, legibility, and contrast. Through this project, I explored the 8 fundamental typographic design systems that enable designers to break free from traditional horizontal and vertical grid constraints.

The Axial system organizes text along a single axis, creating a dynamic flow while maintaining readability.
